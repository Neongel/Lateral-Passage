# 横向通道 (Side Runner)

一款基于 pygame 开发的 2D 横板闯关游戏。

## 功能特点

- 🎮 **内置关卡** - 包含多个精心设计的游戏关卡
- 🗺️ **地图编辑器** - 支持创建自定义关卡，支持鼠标中键拖动和滚轮缩放
- 📤 **地图导入导出** - 关卡文件以 JSON 格式保存，支持分享和备份
- 🌍 **多语言支持** - 支持简体中文和 English
- 🚪 **可交互物品** - 包含门、陷阱、起点、终点等多种游戏元素

## 安装要求

- Python 3.10+
- pygame 2.6+
- lupa (Lua 脚本支持)

## 安装步骤

```bash
# 安装依赖
pip install -r requirements.txt

# 运行游戏
python main.py
```

## 游戏操作

### 游戏中
- **方向键 / WASD** - 移动
- **空格 / 上方向键 / W** - 跳跃
- **ESC** - 返回菜单

### 地图编辑器
- **左键点击** - 放置物品
- **右键点击** - 移除物品
- **鼠标中键拖动** - 平移视图
- **鼠标滚轮** - 缩放视图
- **ESC** - 返回菜单

## 物品说明

| 物品 | 说明 |
|------|------|
| 地面 | 玩家可以站立的平台 |
| 起点 | 玩家开始位置 |
| 终点 | 到达终点完成关卡 |
| 陷阱 | 触碰会导致玩家死亡 |
| 门 | 点击可开关，关闭时无法通过 |
